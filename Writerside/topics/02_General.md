# Thư Mục Chính

Thư mục chính của MagicPack nằm tại **plugins/MagicPacks** và khi truy cập sẽ có nhiều thư mục con. Mỗi thư mục con sẽ có các nhiệm vụ nhất định
![image_4.png](image_4.png)

| Tên File / Thư Mục | Công Dụng                                                                                                                             |
|--------------------|---------------------------------------------------------------------------------------------------------------------------------------|
| assets             | Thành phần assets chính cấu thành nên ResourcePack chứa mô tả và logo của gói ResourcePack **(Không nên đụng vì đã cài đặt sẵn hết)** | 
| builds             | Chứa các gói ResourcePack đã xây dựng xong, có thể vào để tải về đưa cho Member nếu member cài không được!                            | 
| fonts              | Dùng để tạo font image bitmap tức là biến kí tự bất kì thành hình ảnh, icon để dùng làm custom hub jau icon trên scoreboard           | 
| items              | Thư mục quan trọng nhất nơi tạo và lưu trữ model liên quan đến vật phẩm, sẽ thao tác nhiều ở đây                                      | 
| models             | Thư mục quan trọng thứ hai nơi dùng để thêm entity model vào máy chủ, chỉ cần ném file model vào là xong                              |
| sounds             | Nơi dùng để thêm âm thanh, âm nhạc vào server với định dạng .ogg                                                                      |
| config.yml         | Không nên đụng vì đã cấu hình sẵn hết và không cần phải làm gì                                                                        |
