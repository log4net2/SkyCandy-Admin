# Lệnh Chính
| Lệnh                  | Thông Tin                         |
|-----------------------|-----------------------------------|
| /magicpack build      | Xây dựng nhanh gói ResourcePack   | 
| /magicpack all        | Xem toàn bộ vật phẩm custom       | 
| /magicpack gui        | Xem vật phẩm custom theo danh mục | 
| /magicpack font       | Xem danh sách custom font image   | 
| /magicpack sound      | Xem danh sách âm thanh custom     |
| /magicpack spawn <id> <br/>| Triệu hồi entity model với id     |
