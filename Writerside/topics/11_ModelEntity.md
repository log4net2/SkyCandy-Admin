# Model Entity 3D
## Lưu Ý
- Model Entity cũng sử dụng định dạng `.bbmodel` nên không cẩn phải chuyển đổi gì. Chỉ việc ném lên
## Tạo Thư Mục & Ném File
- Để thêm Model Entity thì ta tiến hành tạo thư mục hoặc ném tệp `.bbmodel` lên tại `plugins/MagicPacks/models` 
![image_35.png](image_35.png)

## Xây Dựng Gói Pack
- Khi đã ném tệp `.bbmodel` xong thì ta vào game và chạy lệnh **/magicpack build**
- Ta có thể triệu hồi Model Entity vừa thêm bằng lệnh **/magicpack spawn <tên file>**
