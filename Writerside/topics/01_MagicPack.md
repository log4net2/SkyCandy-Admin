# MagicPack

MagicPack được kế thừa từ SkyCandyPack được phát triển bởi **log4net** với nững mục tiêu:

* Thêm Model Item, Armor.. (Tạo item mới)
* Thêm Model Entity (Tạo entity mới)
* Thêm Custom Sound (Tùy chỉnh âm thanh)
* Thêm Custom Font Image (Thêm hình ảnh vào game)
